# Video_Downloader
A little YouTube videos downloader.

### Requirements:
<p> -Pytube. </p>

### Install requirements:
<p> $ pip install pytube </p>

### How to use:
<p> You just need to put the link of the video. When the video has been downloaded, it will appear in your download directory.</p>
